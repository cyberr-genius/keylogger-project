Online Keylogger – README
⚠️  Warning

This tool is provided for educational, testing, and cybersecurity research purposes only.
Using a keylogger without the explicit consent of the individuals involved may be illegal in many jurisdictions and is a serious violation of privacy.

 You are solely responsible for how you use this tool.


This project aims to:

Understand how keystroke logging systems work
Test application and system security
Raise awareness about the risks of unauthorized monitoring

This tool may be used legally in the following scenarios:
Authorized security testing (pentesting) with written consent
Learning environments or lab simulations
Parental control with clear and informed consent
